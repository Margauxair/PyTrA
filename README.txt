=====
PyTrA
=====

PyTrA is a program to analysis ultrafast transient absorption spectroscopy data.

In order to install it is recommended that you install the Enthought academic distribution.

You can then use easy_install program from the command line (command prompt) e.g. >>easy_install pymc

There packages that need to be installed separately are:

* pymc - for MCMC fitting
* pymodelfit - for fitting of the data
* pywin32 - Needed for windows if you want to use Igor Pro from PyTrA
